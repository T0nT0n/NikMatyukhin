# NikMatyukhin

<p>Эта программа находит Наименьшее общее кратное (НОК) и Наибольший общий делитель двух введённых чисел, а также сравнивает введённые числа.</p> 
<br> 
<h3>Функции программы</h3> 
<ul> 
<li>Вычисление НОК двух чисел.</li> 
<li>Вычисление НОД двух чисел</li> 
<li>Сравнение чисел</li> 
</ul>
